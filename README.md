# Personal Portfolio Website

## Overview

This project is my personal portfolio website, created as part of the **Advanced Software & Web Developer Diploma** at **Pitman Training**. The site showcases my background, technical skills, and academic projects while demonstrating my understanding of front-end web development fundamentals.

The portfolio is built using **HTML, CSS, and JavaScript** and follows a clean, responsive, single-page layout.

---

## Technologies Used

- HTML5  
- CSS3  
- JavaScript (Vanilla)  
- Formspree (contact form handling)  
- GitHub (version control and hosting)

---

## Features

### Navigation
- Fixed navigation bar with smooth scrolling
- Links to Home, About, GitHub, and Contact sections
- CSS hover effects for improved usability

### Home Section
- Full-screen background video using HTML5
- Overlay text introduction
- CSS filters applied for readability

### About Section
- Overview of my background and current studies
- Highlights my focus on C#, .NET, MVC, databases, and web development
- Responsive two-column layout using Flexbox

### GitHub Section
- Direct link to my GitHub profile
- Description of academic and coding projects

### Contact Section
- Popup contact form controlled with JavaScript
- Form submissions handled via Formspree
- Click-outside detection to close the form

### Footer
- Automatically displays the current year using JavaScript

---

## CSS & JavaScript Highlights

- Flexbox used for layout and responsiveness
- Transitions and hover effects for user interaction
- JavaScript handles form popups, event listeners, and dynamic content
- Clean separation of structure, styling, and behavior

---

## Purpose

This project demonstrates:
- Front-end development fundamentals
- Responsive design techniques
- JavaScript DOM manipulation
- Practical application of skills gained during the diploma
